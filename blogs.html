Local Storage vs Session Storage

local Storage: Data stored in local storage never expires, and it has a capacity of 10mb. It stores data in the browser. Session Storage: Data stored in the session storage expires after the window is closed. It has a lesser capacity than local storage of 5mb. It also stores data in the browser.

Javascript event loop

Javascript is a synchronus programming language, but it can achieve asynchronus capability using the browsers help.Event loop , call stack, and the browser works together to make it asyncronus. When there is a function in call stack the event loop will wait and store the other functions and when the call stack is empty it will push the next function in line for execution.

How many ways can we get undefined

1) Variable that is not initialized will give undefined.
2) Function with no return
3) Parameter that is not passed will be undefined
4) If return has nothing on the right side
5) Property that doesn't exist in an object will give you undefined
6) Accessing array elements outside of the index range
7)deleting an element inside an array
8)Set a value direclty to undefined
